# Manual Testing Project – SauceDemo

## Project Overview
This repository contains manual testing artifacts for the SauceDemo web application.
The project focuses on validating core functionalities such as login, product selection, cart, and checkout process.

## Application Under Test
https://www.saucedemo.com

## Testing Scope
* Login functionality
* Product listing
* Add to cart
* Checkout process
* Form validations
* Error message validation

## Testing Types
* Manual Testing
* Functional Testing
* UI Testing
* Regression Testing

## Repository Contents

### Test Cases
Contains manual test cases written in Excel format.

### Bug Reports
Contains bug reports with severity, priority, and reproduction steps.

### Test Scenarios
Contains high-level testing scenarios for different modules.

## Sample Bugs Identified
* Checkout button enabled with empty cart
* Invalid ZIP code accepted during checkout
* Incorrect product images displayed for specific users
* UI formatting inconsistencies in product descriptions
* Slow login observed for performance_glitch_user
